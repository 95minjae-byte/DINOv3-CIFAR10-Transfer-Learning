# Transfer Learning with DINOv3 for CIFAR-10 Image Classification

This repository contains the code used for the final project:

- DINOv3 + Linear Classifier
- Random ViT + Linear Classifier
- CNN Baseline

## Dataset

CIFAR-10

## Training Configuration

- Optimizer: Adam
- Learning Rate: 0.001
- Batch Size: 32
- Epochs: 5

## Results

| Model | Accuracy |
|---------|---------|
| Random ViT | 28.51% |
| CNN | 70.79% |
| DINOv3 | 90.19% |
